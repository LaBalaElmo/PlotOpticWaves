Plot Functions

In this page you can plot multiples functions at the same time, only the function field is required and the rest of the fields are just fields to improve how the graphs look.

The function field must be filled like the following example:

Each function must be separated by a comma (,) and the functions must be functions of x. Then you can just press the plot button or fill the other fields. If you press the button plot, a graph with all the functions will appear like this:

However, you can fill all the other fields that will give to the graph a better look. First, the graph limit on x-axis field, by default it goes from -10 to 10, but for most graphs it may be too much. The syntax for this field is two numbers separated by a comma (,). Similarly, the graph limit on y-axis field has the same syntax, but this last field doesn't always work, it only works when the limit function is smaller than the limit given by the previous field.
